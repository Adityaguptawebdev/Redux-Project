
# Redux Practice Project

This project is built to learn and practice **Redux Toolkit** with **React**.  
It focuses on understanding global state management, actions, reducers, and store configuration.

## Tech Stack
- React
- Redux Toolkit
- React Redux
- Axios
- Tailwind CSS

## What I Learned
- Setting up Redux store
- Creating slices using Redux Toolkit
- Managing global state
- Dispatching actions from components
- Connecting Redux with APIs

This project is purely for learning and experimentation.
